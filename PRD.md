## 2. 제품 개요

2026 영남공업고등학교 학생들을 위한 교내 통합 어플리케이션 ‘영공이’ 기획안이다.

## 3. 목표

- 학교 생활에 필요한 기능을 한 앱에서 제공한다.
- 학생 관리, 커뮤니티, 학사 정보, 사용자 편의 기능, 출결 및 보안 기능을 제공한다.

## 4. 대상 사용자

- 2026 영남공업고등학교 학생
- [확인 필요: 교사도 사용 대상인지]
- [확인 필요: 학부모도 사용 대상인지]

## 5. 기능 요구사항

### 5.1 학생 관리 및 인증

- 학교 학생 인증 시스템이 필요하다.
- 상벌점 현황을 제공한다.

### 5.2 커뮤니티

- 전체 익명 채팅을 제공한다.
- 학년별 익명 채팅을 제공한다.
- 학교 공지와 이벤트를 제공한다.
- 온라인 가정통신문과 공지사항을 제공한다.
- 급식 리뷰 기능을 제공한다.

### 5.3 학사 정보

- 시간표를 제공한다.
- 급식 정보를 제공한다.
- D-day를 제공한다.
- 가까운 학교 행사 알림을 제공한다.
- 동아리 홍보 및 신청 기능을 제공한다.
- 호림재 예약 기능을 제공한다.

### 5.4 사용자 편의 및 활동

- 마이페이지를 제공한다.
- 마이페이지에는 익명 프로필을 사용한다.
- 레벨 시스템을 제공한다.
- 뱃지 시스템을 제공한다.
- 감정버튼(리액션) 기능을 제공한다.
- 공부시간 카운트 기능을 제공한다.
- 시간표, D-day, 급식 위젯 설정을 제공한다.

### 5.5 출결 및 보안

- BLE 기반 등교 시간 기록 기능을 제공한다.
- 교문 앞 태블릿 거치 인증을 제공한다.
- 스마트폰 미지참자를 위한 인증 수단을 제공한다.

## 6. 기술 스택

### 6.1 Firebase

- 유저 인증
- Push 알림
- Cloud Storage
- Hosting
- 커뮤니티(Firestore)

### 6.2 VPS 서버

- Main API Server
- Socket.io 기반 실시간 채팅
- Database
- Redis
- RFID/BLE 관련 로직

## 7. 구현 전략

### 7.1 출결 시스템

- RFID의 물리적 제약을 극복하기 위해 BLE(Bluetooth Low Energy) 방식을 채택한다.
- 스마트폰 미지참 학생을 위한 교문 앞 태블릿 인증 시스템을 병행한다.

### 7.2 급식 정보

- NEIS API를 활용하여 자동 연동한다.

### 7.3 시간표 관리

- 학교별 고유 시간표 지원 API 부재로, 담당 교사 직접 수정 방식의 관리 도구를 구현한다.

### 7.4 채팅 서비스

- 비용 효율성과 복잡도를 고려하여 Firebase 대신 Socket.io(Node.js 기반)로 구현한다.

### 7.5 커뮤니티 기능

- Firebase Firestore 및 Storage를 활용하여 글과 사진 업로드를 지원한다.
